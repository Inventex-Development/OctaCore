# OctaCore
A general purpose utility framework for java projects

## Installation
```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```

```xml
<dependencies>
    <dependency>
        <groupId>com.github.Inventex-Development</groupId>
        <artifactId>OctaCore</artifactId>
        <version>1.0.6</version>
    </dependency>
</dependencies>
```
